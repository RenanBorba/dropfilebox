Aplicação desenvolvida para o clone da versão web do Dropbox, voltada para serviço de armazenamento de arquivos, permitindo, assim, a atualização em tempo real dos arquivos recém enviados via WebSocket.

## :arrow_forward: Start
### Node.js

</ul>
